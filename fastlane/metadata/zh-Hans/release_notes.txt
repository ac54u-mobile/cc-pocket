本次更新（1.4.0）

新功能
• 打开项目：手机上浏览电脑目录，任选一个文件夹直接起新会话，不再只能从历史会话进入
• 切模型更顺手：输入框旁新增模型徽标，点一下直达模型选择器（手机与桌面端）
• 桌面端内嵌终端：⌘J 呼出停靠终端面板，会话可选「在 App 内打开」，默认方式可在设置里改
• macOS 菜单栏状态胶囊：常驻图标一眼看清连接与回合状态（五态），点击弹出快捷面板
• 桌面端会话重命名：侧栏右键内联改名，名称与 Claude Code 自家记录同步
• Bridge 受限凭证（预览）：以最小权限把 Claude Code 接入外部机器人（附飞书群参考适配器），手机与桌面端可监控和吊销

修复与优化
• 首次配对更可靠：配对首连被网络波动打断后，之前可能一直连不上且重新扫码也无效（需重启电脑端），现在重连即自动恢复
• iOS 任务推送修复：部分安装收不到完成/审批推送的问题已解决
• 消息里的链接不再把尾部括号误吞进网址，点开即达
• bypassPermissions 模式下，计划确认不再被自动放行，仍会等你点头

提示：部分功能需要电脑端 cc-pocket-daemon 同步更新到 1.4.0——运行 cc-pocket-daemon update（或 brew upgrade / scoop update）。
